# Instruções básicas

- Todos os arquivos nesta pasta serão de instruções sobre como gerar o conteúdo que preciso para as aulas. Siga cada pedido nos arquivos com atenção, gerando um conteúdo bastante detalhado sobre o que é pedido.
- Use sempre lingua portuguesa.
- Gere o conteúdo de saída em formato Markdown e escreva em um arquivo na pasta "content_documents", com o mesmo nome do arquivo que o gera vindo da pasta "requests_prompts".
- Quando houver pedido para Figura, estará entre parênteses: "(gerar figura)". Siga a instrução do tópico em que o pedido se encontra cautelosamente para gerar a imagem, salve a mesma na pasta "images". Adicione no texto final do arquivo Markdown a citação e carregamento da figura da maneira "Figura X", segundo a legenda da figura que você pode colocar de forma a casar com o conteúdo da imagem e do tópico que a requisitou.
- Cada tópico pedido deve gerar um capítulo do Markdown de saída. 
- Caso não tenha conhecimento sobre datas ou lançamentos recentes após a sua data de treinamento, faça uma busca online sobre o conteúdo.
- Cada arquivo de prompt de entrada deve gera um arquivo de conteúdo em Markdown.
